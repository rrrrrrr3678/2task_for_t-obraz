# 2 задание для проектной смены
Задание выполнено на языке программирования python. Для выполнения задач были использованны следующие библиотеки: json(для хранения книг в отдельном файле и полседующем восстановлении данных оттуда при повтороном запуске), os(для проверки путя к файлу), tkinter(выводит программу на консольное приложение)

В окне с названием "Т-Библиотека" есть раздел со списиском всех книг, ниже него находятся строчки для ввода информации о книге: ее названии и ее авторе. Далее идут кнопки для выполнения действий: добавить книгу, удалить книгу, пометить книгу как прочитанную или непрочитанную, пометить книгу как избранную
